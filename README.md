# MkDocs Preview for VS Code
A focused VS Code extension that spins up a live MkDocs preview for intergrated doc browsing.

## 1. Before you start

### Requirements
- VS Code `1.109.0` or later
- Python 3.8+ with MkDocs installed (`pip install mkdocs`) in either a virtual environment or globally

### Verify MkDocs
```bash
mkdocs --version        # should print the version
```
